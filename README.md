# Sales Performance Analysis

## Objective
The objective of this project is to analyze sales data and identify high-performing regions, customers, and monthly trends.

## Dataset
The dataset contains order-level sales information including:
- Order ID
- Customer
- Region
- Sales amount
- Month

## Tools Used
- Python
- Pandas
- SQL logic (group by, aggregation, filtering)

## Analysis Performed
- Total sales by region
- Month-wise sales analysis
- Customer-wise sales contribution
- Identification of high-value customers

## Key Insights
- The North region generated the highest total sales.
- February recorded higher sales compared to January.
- Customer A contributed the highest share of total revenue.

## Business Recommendation
Focus on high-performing regions and retain high-value customers through targeted offers and loyalty programs.
